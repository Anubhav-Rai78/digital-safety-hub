# Project Aegis: Digital Safety Hub
An immersive, cyber-safety platform project.

### Core Features
- **Aegis AI Guardian**: Real-time scam analysis using Gemini.
- **Persistent 3D UI**: Immersive Three.js environment that never reloads.
- **Safe reporting**: Direct integration for reporting digital fraud.

### Tech Stack
- **Frontend**: Three.js, Vanilla JS (SPA Architecture), CSS Grid/Flex.
- **Backend**: FastAPI (Python), Supabase (Database).
- **AI**: Google Gemini API.
